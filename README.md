# dotfiles

My various system setup files and configurations.

## MacOS

Install with homebrew:

```bash
brew install $(cat brew-packages.txt)
```

## Fetching configs

```sh
wget https://raw.githubusercontent.com/asgarli/dotfiles/refs/heads/master/.aliases
wget https://raw.githubusercontent.com/asgarli/dotfiles/refs/heads/master/.vimrc
```
